With this plugin you can place an iframe inside the TinyMCE text editor.

== Description ==

With this plugin you can place an iframe inside the TinyMCE text editor.
You click a button in TinyMCE and a window appears to place the iframe data.
Then just click on Ok and a shortcode is inserted with the iframe data and on the front the iframe is presented.

Com este plugin você consegue colocar um iframe dentro do editor de texto TinyMCE.
Voce clica em um botão no TinyMCE e aparece uma janela para colocar os dados do iframe. 
Depois é só clicar em Ok e um shortcode é inserido com os dados do iframe e no front é apresentado o iframe.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/rafa-tinymce-iframe` directory, or install the plugin through the WordPress plugins screen directly.
